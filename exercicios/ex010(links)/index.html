Vamos criar um link

  1. target=" "

    • _blank - aabre o link em uma nova janela;
    • _self - abre o link na janela atual;
    • _top - desfaz todos os frames e abre o destino no navegador completo;
    • _parent - similar ao top em uma referencia à janela mãe

    O site abre na mesma aba(_self, padrão)

    YouTube

    O site abre em uma nova aba(_blank)

    w3schools

  2. Atributo rel=" " indica a natureza do destino

    • self- continuar navegando no site;
    • next- indica que o link é para a próxima parte do documento atual;
    • prev- indica que o link é para a parte anterior do documento atual;
    • author- indica que é um link paraa o site do autor do artigo atual;
    • external- indica que é um link para outro site;
    • nofollow- indica que é um link para um site não endossado, como um link pago.

  3. Para fazer downloads

    4. <a href="link do download" download="nome do arquivo.extenção" type="tipo.extenção"

    Media types mais usados:

    • application/zip
    • text/html ou /css ou /javascript
    • video/mp4 ou /H264 ou /JPEG
    • audio/aac ou /mpeg
    • font/ttf
    • image/jpeg ou /png
